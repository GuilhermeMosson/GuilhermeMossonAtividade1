Tecnologias

HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Ele utiliza uma série de tags para definir diferentes partes do conteúdo, como cabeçalhos, parágrafos, links, imagens, etc.

CSS

Cascading Style Sheets é um linguagem de estilo utilizada para descrever a apresentação de uma página web. Com o CSS, você pode definir o layout, cores, fontes, tamanhos, e outros aspectos visuais de um documento web.

JavaScript

JavaScript é uma linguagem de programação orientada a objetos, permitindo interações dinâmicas dentro das páginas da web. Atualmente o JavaScript se tornou uma das linguagens de programação mais populares, sendo amplamente utilizada em aplicativos móveis, servidores, jogos e muito mais.
